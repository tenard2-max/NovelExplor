# NovelExplor Workspace (XML DB)

GitHub Pages: https://tenard2-max.github.io/NovelExplor/

## 구조

```
data/workspace/
  workspace.xml
  sections/          ← 뷰별 캔버스 XML (reader ≠ character)
  assets/
    characters/CHR0001/avatar.png   ← 테스트 PNG
    stories/ST999.md                ← 테스트 MD
```

## Pages 테스트

1. 인물 → CHR0001 카드에 파란 테스트 PNG
2. 소설 읽기 → 드롭다운 **제999화** → ST999.md 본문
3. 마스터/복선/타임라인 → 각 섹션 XML 배너 경로 확인
